# Getting started

Install Comfy Deploy's plugin on your local ComfyUI to get started with deploying workflow.

<CodeGroup>

```bash git
cd custom_nodes
git clone https://github.com/BennyKok/comfyui-deploy.git
```

```js ComfyUI Manager
// Install with ComfyUI Manager
Search`ComfyUI Deploy`;
```

</CodeGroup>
